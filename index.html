What is Cortogen?

  • Long-Term Semantic Memory: A lightweight browser extension that connects ChatGPT, Claude, and Gemini with a local semantic memory database.
  • Context Auto-Retrieval: Automatically indexes conversation turns and surfaces matching past context in real time to prevent starting chats from scratch.
